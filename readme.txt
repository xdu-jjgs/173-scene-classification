三、其他说明
这个项目由于两种数据集的图片大小不一样，一个是32*32，一个是256*256的，所以没法目前共用一个模型，其实做一下什么数据预处理的技巧也可以共用但是我懒。。。
然后目前用的模型是一个rnn模型，如果你对这个模型感兴趣，找我要论文，或者你直接看代码看能不能看懂，我估计反传那块比较难看懂。它是RACNN算法上进行了化简得到的，也是赵宁师姐的工作里用到的一个小部分，缺点是训练特别慢，但推理速度符合要求，优点是相对参数较小。
